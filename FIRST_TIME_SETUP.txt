================================================================================
  Flowxtra SERP Tracker - Multi-Tenant Edition
  دليل الإعداد الأول
================================================================================

هذا الملف يحتوي على جميع الأوامر المطلوبة لإعداد النظام من الصفر.
اتبع الخطوات بالترتيب.

================================================================================
الخطوة 1: تثبيت الحزم المطلوبة
================================================================================

في terminal، في مجلد المشروع، قم بتشغيل:

npm install


================================================================================
الخطوة 2: إعداد MySQL Database
================================================================================

2.1 - افتح MySQL Command Line:

mysql -u root -p


2.2 - أنشئ قاعدة البيانات:

CREATE DATABASE flowxtra_serp CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'flowxtra_user'@'localhost' IDENTIFIED BY 'YourSecurePassword123';
GRANT ALL PRIVILEGES ON flowxtra_serp.* TO 'flowxtra_user'@'localhost';
FLUSH PRIVILEGES;
USE flowxtra_serp;
EXIT;


أو استخدم SQL file الجاهز:

mysql -u root -p < database/setup.sql


================================================================================
الخطوة 3: تكوين Environment Variables
================================================================================

3.1 - انسخ .env.example إلى .env.local:

# في Windows:
copy .env.example .env.local

# في Linux/Mac:
cp .env.example .env.local


3.2 - افتح .env.local وحدّث القيم التالية:

# Database Configuration
DB_DIALECT=mysql
DB_HOST=localhost
DB_PORT=3306
DB_NAME=flowxtra_serp
DB_USER=flowxtra_user
DB_PASSWORD=YourSecurePassword123

# إذا كنت تريد استخدام SQLite للتطوير:
# DB_DIALECT=sqlite
# (باقي DB_* variables يمكن حذفها أو تركها)


================================================================================
الخطوة 4: تشغيل Database Migrations
================================================================================

npm run db:migrate


ستحصل على output مثل:
✓ Users table created successfully
✓ user_id added to domain table successfully
✓ user_id added to keyword table successfully


إذا حدث خطأ، يمكنك التراجع:

npm run db:revert


ثم حاول مرة أخرى:

npm run db:migrate


================================================================================

================================================================================

الطريقة 1 - عبر SQL (سريعة):

mysql -u flowxtra_user -p flowxtra_serp

ثم:

بيانات الدخول:
Email: admin@flowxtra.com

الطريقة 2 - عبر صفحة التسجيل:

1. شغّل التطبيق: npm run dev
2. افتح: http://localhost:3000/register
3. املأ البيانات وسجّل


================================================================================
الخطوة 6: تشغيل التطبيق
================================================================================

npm run dev


افتح المتصفح على:
http://localhost:3000


================================================================================
الخطوة 7: تسجيل الدخول
================================================================================

انتقل إلى: http://localhost:3000/login

إذا استخدمت الطريقة 1 أعلاه:
Email: admin@flowxtra.com
Password: admin123

إذا سجّلت عبر صفحة التسجيل:
استخدم البريد وكلمة المرور التي أدخلتها


================================================================================
الخطوة 8: التحقق من النظام
================================================================================

بعد تسجيل الدخول:

1. اذهب إلى: http://localhost:3000/domains
2. جرّب إضافة domain جديد
3. أضف keywords
4. تحقق من أن البيانات تظهر بشكل صحيح


================================================================================
استكشاف الأخطاء الشائعة
================================================================================

❌ خطأ: "Cannot connect to database"
   الحل: تحقق من أن MySQL يعمل
         تحقق من username/password في .env.local
         تحقق من اسم database

❌ خطأ: "Table users doesn't exist"
   الحل: قم بتشغيل: npm run db:migrate

❌ خطأ: "Module not found: mysql2"
   الحل: قم بتشغيل: npm install

❌ خطأ: "Invalid email or password"
   الحل: تأكد من البريد الإلكتروني صحيح
         أو استخدم الحساب الافتراضي: admin@flowxtra.com / admin123

❌ خطأ: "Port 3000 already in use"
   الحل: أوقف التطبيقات الأخرى على port 3000
         أو غيّر PORT في command: PORT=3001 npm run dev


================================================================================
الأوامر المفيدة
================================================================================

# تطوير
npm run dev              # تشغيل development server
npm run build            # بناء للإنتاج
npm start                # تشغيل production server

# Database
npm run db:migrate       # تطبيق migrations
npm run db:revert        # التراجع عن migration

# مع CRON Jobs
npm run cron             # تشغيل CRON jobs فقط
npm run start:all        # تشغيل server + cron معاً

# Testing
npm test                 # تشغيل tests
