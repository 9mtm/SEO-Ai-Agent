# نظام الألوان والأزرار الموحد

## نظرة عامة
تم تحديث نظام الألوان في التطبيق ليكون موحداً وجميلاً عبر جميع الصفحات.

## الألوان الأساسية

### Primary (اللون الأساسي)
- **اللون**: أزرق حيوي `hsl(221.2 83.2% 53.3%)`
- **الاستخدام**: الأزرار الرئيسية، الروابط المهمة
- **CSS Variable**: `--primary`

### Secondary (اللون الثانوي)
- **اللون**: رمادي فاتح `hsl(210 40% 96.1%)`
- **الاستخدام**: الأزرار الثانوية، الخلفيات
- **CSS Variable**: `--secondary`

### Destructive (لون التحذير)
- **اللون**: أحمر `hsl(0 84.2% 60.2%)`
- **الاستخدام**: أزرار الحذف، التحذيرات
- **CSS Variable**: `--destructive`

## أنواع الأزرار

### Default (الافتراضي)
```tsx
<Button>Click Me</Button>
```
- خلفية زرقاء مع نص أبيض
- ظل متوسط يزداد عند التمرير
- تأثير scale عند الضغط

### Outline (محدد)
```tsx
<Button variant="outline">Click Me</Button>
```
- خلفية شفافة مع حدود
- يتغير اللون عند التمرير

### Secondary (ثانوي)
```tsx
<Button variant="secondary">Click Me</Button>
```
- خلفية رمادية فاتحة
- مناسب للإجراءات الثانوية

### Destructive (تحذيري)
```tsx
<Button variant="destructive">Delete</Button>
```
- خلفية حمراء للإجراءات الخطرة

### Ghost (شبح)
```tsx
<Button variant="ghost">Click Me</Button>
```
- بدون خلفية، يظهر عند التمرير

## الأحجام

- **sm**: صغير `h-9`
- **default**: متوسط `h-10`
- **lg**: كبير `h-11`
- **icon**: للأيقونات `h-10 w-10`

## التأثيرات

### Hover (التمرير)
- تغيير طفيف في اللون (`/90`)
- زيادة الظل
- انتقال سلس

### Active (الضغط)
- تأثير scale صغير `scale-[0.98]`
- يعطي إحساساً بالضغط الفعلي

### Focus (التركيز)
- حلقة زرقاء للوصولية
- offset للوضوح

## الاستخدام في الصفحات

تم تحديث الصفحات التالية:
- ✅ صفحة تسجيل الدخول
- ✅ صفحة التسجيل
- ✅ الصفحة الرئيسية
- ✅ صفحة الإعدادات
- ✅ مكون Button الأساسي

## ملاحظات التطوير

### استخدام المتغيرات
استخدم دائماً متغيرات CSS بدلاً من الألوان المباشرة:
```tsx
// ✅ صحيح
className="bg-primary text-primary-foreground"

// ❌ خطأ
className="bg-blue-600 text-white"
```

### التناسق
جميع الأزرار الآن تستخدم:
- `rounded-lg` للحواف
- `font-semibold` للنص
- `transition-all` للانتقالات
- `shadow-md` للظل الأساسي
- `active:scale-[0.98]` لتأثير الضغط
